# Security Policy

## Scope

This repository contains a locally deployed Docker application with LLM API integration. Security concerns may involve:
- Sensitive information (API keys, credentials) inadvertently committed
- Vulnerabilities in application code or dependencies
- LLM prompt injection or output handling issues

## Reporting a Vulnerability

If you discover a security concern in this repository, please report it responsibly.

**Do not open a public GitHub issue for security or privacy concerns.**

To report:

1. Open a [GitHub Security Advisory](https://github.com/wtlinnertz/aieos-console/security/advisories/new) in this repository, or
2. Contact the maintainers directly via the repository's contact information.

Include:
- A clear description of the concern
- The file(s) affected
- Steps to reproduce or locate the issue, if applicable

We will acknowledge reports within 48 hours and address confirmed issues promptly.

## Supported Versions

Only the current `main` branch is supported.
